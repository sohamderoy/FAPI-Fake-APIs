# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-16

### Added
- Endpoint URL preview in create and edit modals for better visibility
- Enhanced URL copy functionality with improved user experience
- CLI consent pre-startup script for better user onboarding

### Changed
- Minor styling improvements in create and edit modal

## [1.0.0] - 2026-01-15

- Quick setup with intuitive UI for building custom API endpoints
- Zero configuration file-based storage with no database required
- 100% local and private operation with offline support
- Persistent storage for endpoints and configurations across sessions
- Customizable HTTP status codes and JSON responses
- Response delay simulation for realistic network latency testing
- Support for GET, POST, PUT, DELETE HTTP methods
- Create, edit, and delete endpoints on the fly
- Multiple FAPI instances on different ports with isolated data
- Import/Export functionality for sharing API configurations
- Project names for organizing different FAPI instances
- Version logging for debugging and support
- Privacy banner in simulator page
- FAPI logo and favicon
